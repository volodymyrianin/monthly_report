VAG rework

General information

In August the global VAG rework was started. The main goal is to add all the engine codes and redo the engines across VAG.

What have been done?

Volkswagen Golf MK2, Volkswagen Golf MK3, Volkswagen Jetta MK2 were rewritten in AMR. All the engines in those generations were renamed according to the official brochures, years altered and engine codes added where it was needed.

Current status

Right now, Golf MK3 was rewritten according to the model codes that we were able to gather. However, there are still rooms for improvement. Currently, looking for missing model years and gearboxes while checking the connected idents. Moreover, there are some problems with some model codes` patterns. Seems like some of them were filled in by the government data operators not 100% accurate which causing conflicts in rules. Still, it seems like using model codes is the best way of writing the rules for the older VW model, as soon as door count data is not always available and gross weight data does not differ depending on the chassi.

Plans for the next month

The main focus right now is on Golf model and at the same time on the cross models that use the same engines as Golf. During the next month Golf MK4 would be under the main focus.
