VAG rework

General information

We continue our work with VAG, for now only with Volkswagen. It is quite long going process considering the amount of information and various variables/

What has been done?

During October Volkswagen Golf MK3 facelift and Caddy Typ 2K 2nd Facelift were rewritten. The first one was rewritten according to the plan. AS for the second one, it was urgent request ;) However, it showed that the rewriting process for the latest models would be way easier. The reason for it is that those models have some kind of logical model codes pattern.

Current status

Right now it is a time of Golf MK4 ;) almost all the missing engines are added or differentiated. SO, almost everything is ready for the initial AMR rules writing.

Plans for the next month

Main focus is going to be on Golf MK4. If nothing urgent happens there is a possibility that during November MK4 would be done and some preparations for the next generations would be done too :)
